SIMPLE API

1) http://127.0.0.1:8000/data1 -->GET Method to receive all data

2) http://127.0.0.1:8000/data/4 -->GET Method to receive data by id

2) http://127.0.0.1:8000/create_data -->POST Method to create data

2) http://127.0.0.1:8000/update_data-->PUT Method to update all field data (by using partial=true we can update data partially without using patch method)

2) http://127.0.0.1:8000/delete-->DELETE Method to DELETE data (need to pass only id)

The below dat is used to create update
{ "id":"1", "user":"112", "mobile": "333d", "image":"rrrr", "gender": "eeee", "type": "fkfff", "resume":"rlllllll" }
The below dat is used to delete data
{ "id": "4" }
